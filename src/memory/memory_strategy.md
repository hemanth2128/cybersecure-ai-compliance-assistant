Session Memory:
- Holds temporary conversation-based organization info.

Long-term Memory:
- Stores previously analyzed organizations
- Re-uses context if the same organization returns

Memory Format Example:

{
 "organization": "SecurePay Fintech",
 "industry": "Finance",
 "cloud": "AWS",
 "previous_risks": [...],
 "last_assessment_date": "2025-11-28"
}
